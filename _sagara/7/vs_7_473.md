प्राणायामः आसन सिद्ध्यनन्तरं प्राणायामः कर्तव्यः।
प्राणायामस्त्वनेकै विधः।
तथापि तल्लक्षणं सङ्ग्रहेणोच्यते ( १ ) नासिकायाः वामच्छिद्रस्थेडाख्यनाडीद्वारा वायोरन्तः पूरणं कार्यम्।
तदेव पूँरकमित्युच्यते (२) दक्षिणनासिकास्थपिङ्गलाख्यनाडीद्वारा वायोर्बहिः निःसरणं कार्यम्।
तदेव रेचकमित्युच्यते ( ३ )
सुषुम्नाद्वारा वायोर्निरोधनं कार्यम्।
तदेव कुम्भकमित्युच्यते।
इत्थं क्रियमाणानां

१. चित्तादिसर्वभावेषु ब्रह्मत्वेनैव भावनात्।

२.
निरोधः सर्ववृत्तीनां प्राणायामः स उच्यते ॥
केचन प्राणायाम ( 1 ) बाह्यं ( २ ) आभ्यन्तरम् ( ३ ) स्तम्भवृत्तिकमिति
निश्वासं कृत्वा तद्गतिनिरोधो बाह्यः।
प्राणोच्छ्वासं कृत्वा तन्निरोधः
आभ्यन्तरः।
बाह्यान्तर्गतेः समकालं निरोधः स्तम्भवृत्तिरिति ॥ इति च।
वदन्ति।

३. अहं ब्रह्मास्मीति भावना पूरकः।

४.

५.
42
अनात्मप्रपञ्च निषेधो रेचकः।
अखण्डाकारवृत्याचलता कुम्भकः।
पूरकरेचककुम्भकानां समुदायः प्राणायामः इत्युच्यते।
स च प्राणायामो
द्विविधः (१) एकोऽगर्भः ( २ ) अपरः सगर्भः।

(१) प्रणवोच्चारणं
विना क्रियमाणः प्राणायामोऽगर्भः (२) प्रणवोच्चारणेन सह क्रियमाणः
प्राणायामः सगर्भः।
